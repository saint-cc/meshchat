Welcome

MeshChat Documentation Hub

This environment contains protocol notes, experimental concepts and small isolated HTML examples used during development of the MeshChat communication system.

The examples intentionally focus on one subsystem at a time:

  • Deterministic identity generation
  • Cryptographic signing and verification
  • Encrypted message exchange
  • Relay synchronization behavior
  • Migration between domains
  • Entropy and identity analysis

Start with Introduction for architecture overview or open the Greypaper for the broader protocol concepts.
