MeshChat

MeshChat is an experimental messaging system focused on portable identity, direct communication and minimal infrastructure trust.

Messages are end-to-end encrypted and signed. Relays transport encrypted packets but cannot read message contents.

There are no centralized accounts or phone-number based identities. A user identity is derived locally from a username and passphrase.

The same credentials recreate the same cryptographic identity on any compatible client.

Contacts are added by exchanging identity information directly between users, for example through QR codes or existing trusted communication channels.

The system prefers direct peer connectivity when available, while fallback transport mechanisms help communication continue during outages or migration.

Experimental protocol and research project.
Current focus areas: identity portability, resilient synchronization and decentralized transport behavior.
